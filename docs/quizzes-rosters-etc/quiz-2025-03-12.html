Question 5.2: What is \(\partial_t A x e^{kt}\)?

  1. \(0\)     (ii) \(A k e^{kt}\)    (iii) \(A k x e^{kt}\)     (iv) \(A e^{kt}\)     

Question 5.3: What is \[\partial_{yx} \left[ a_0 + a_1 x + b_1 y + c x y + a_2 x^2 + b_2 y^2 \right]\ ?\]

  1. \(c\)     (ii) \(2 a_2\)    (iii) \(2 b_2\)    (iv) \(0\)     

Question 5.4: What is \(\partial_{yx} h(x,y) g(y)\)?

  1. \[\partial_{yx} h(x, y)\]
  2. \[g(y) \partial_{yx} h(x,y) + h(x,y)\ \partial_y g(y)\]
  3. \[(\partial_y g(y)) \ (\partial_x h(x, y)) + g(y)\ (\partial_{yx} h(x, y))\]
  4. \[(\partial_x g(y))\ (\partial_x h(x, y)) + g(y) (\partial_{xx} h(x, y) )\]

Question 5.5: What is \(\partial_{t} \left(7 + 8 t^2 + 3 t^4\right)\)?

  3. \(16 t^2 + 9 t^3\)
  4. \(4 t + 12 t^2\)
